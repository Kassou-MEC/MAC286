Introduction to Java Arrays

An array in Java is a container object that holds a fixed number of values of a single type. The length of an array is established when the array is created and cannot be changed.

Key Characteristics of Java Arrays

  • Fixed size - once created, the size cannot be changed
  • Indexed - elements are accessed by their numerical index
  • Homogeneous - all elements must be of the same type
  • Zero-based - the first element is at index 0
  • Object - arrays are objects in Java, even if they hold primitives

Basic Array Example

// Declare and initialize an array of integers int[] numbers = {10, 20, 30, 40, 50}; // Access elements by index System.out.println(numbers[0]); // Output: 10 System.out.println(numbers[3]); // Output: 40 // Get the length of the array System.out.println(numbers.length); // Output: 5 // Change an element numbers[2] = 99; System.out.println(numbers[2]); // Output: 99

Declaring and Initializing Arrays

There are several ways to declare and initialize arrays in Java. The most common approaches are shown below.

Different Ways to Create Arrays

// Method 1: Declare, then initialize int[] numbers; numbers = new int[5]; // Method 2: Declare and initialize in one line int[] numbers = new int[5]; // Method 3: Initialize with values int[] numbers = {10, 20, 30, 40, 50}; // Method 4: Initialize with values (alternative syntax) int[] numbers = new int[]{10, 20, 30, 40, 50};

Default Values

When you create an array without initializing with values, Java assigns default values:

int[] intArray = new int[3]; // Default: [0, 0, 0] double[] doubleArray = new double[3]; // Default: [0.0, 0.0, 0.0] boolean[] boolArray = new boolean[3]; // Default: [false, false, false] char[] charArray = new char[3]; // Default: ['\0', '\0', '\0'] String[] strArray = new String[3]; // Default: [null, null, null]

Iterating Through Arrays

You can use for loops or enhanced for loops to iterate through arrays:

int[] numbers = {10, 20, 30, 40, 50}; // Standard for loop for (int i = 0; i < numbers.length; i++) { System.out.println("Element at index " + i + ": " + numbers[i]); } // Enhanced for loop (for-each) for (int num : numbers) { System.out.println("Element: " + num); }

Multi-dimensional Arrays

Java supports multi-dimensional arrays, which are essentially arrays of arrays. The most common type is the 2D array.

2D Array Examples

// Declare and initialize a 2D array int[][] matrix = { {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }; // Access elements System.out.println(matrix[0][0]); // Output: 1 System.out.println(matrix[1][2]); // Output: 6 System.out.println(matrix[2][1]); // Output: 8 // Get dimensions System.out.println(matrix.length); // Output: 3 (number of rows) System.out.println(matrix[0].length); // Output: 3 (number of columns in first row)

Iterating Through 2D Arrays

int[][] matrix = { {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }; // Nested for loops for (int i = 0; i < matrix.length; i++) { for (int j = 0; j < matrix[i].length; j++) { System.out.print(matrix[i][j] + " "); } System.out.println(); } // Enhanced for loops for (int[] row : matrix) { for (int value : row) { System.out.print(value + " "); } System.out.println(); }

Ragged Arrays

Java supports "ragged" arrays where each row can have a different length:

int[][] ragged = new int[3][]; ragged[0] = new int[2]; // First row has 2 columns ragged[1] = new int[4]; // Second row has 4 columns ragged[2] = new int[3]; // Third row has 3 columns // Initialize with values int[][] ragged = { {1, 2}, {3, 4, 5, 6}, {7, 8, 9} };

ArrayList: Dynamic Arrays

While basic arrays have a fixed size, Java provides the ArrayList class which offers dynamic arrays that can grow or shrink as needed.

ArrayList Basics

import java.util.ArrayList; // Create an ArrayList ArrayList names = new ArrayList<>(); // Add elements names.add("Alice"); names.add("Bob"); names.add("Charlie"); // Access elements System.out.println(names.get(0)); // Output: Alice // Get size System.out.println(names.size()); // Output: 3 // Remove elements names.remove(1); // Removes "Bob" System.out.println(names); // Output: [Alice, Charlie]

Key ArrayList Methods

  • add(element) - Appends the element to the end
  • add(index, element) - Inserts element at specified position
  • get(index) - Returns element at specified position
  • set(index, element) - Replaces element at specified position
  • remove(index) - Removes element at specified position
  • size() - Returns number of elements
  • isEmpty() - Returns true if list is empty
  • contains(element) - Returns true if list contains element

ArrayList vs Array

Arrays

  • Fixed size
  • Primitives or objects
  • Faster for certain operations
  • Less memory overhead
  • Use when size is known and fixed

ArrayList

  • Dynamic size
  • Objects only (uses autoboxing for primitives)
  • Slower for certain operations
  • More memory overhead
  • Use when size may change

Converting Between Arrays and ArrayList

import java.util.Arrays; import java.util.ArrayList; import java.util.List; // Array to ArrayList String[] array = {"Apple", "Banana", "Orange"}; ArrayList list = new ArrayList<>(Arrays.asList(array)); // ArrayList to Array ArrayList fruitList = new ArrayList<>(); fruitList.add("Apple"); fruitList.add("Banana"); fruitList.add("Orange"); String[] fruitArray = fruitList.toArray(new String[0]);
